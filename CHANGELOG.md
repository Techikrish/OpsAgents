# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-16

### Added
- Created 10 production-ready AI agents (5 Cloud + 5 DevOps).
- Core LangGraph state flow execution base.
- Multi-provider LLM integrations (OpenAI, Anthropic, Google GenAI, AWS Bedrock, etc.).
- Multi-mode Human-in-the-Loop interactive approval system (CLI and MCP interrupts).
- FastMCP tool registration server supporting stdio and SSE transport protocols.
- Command-line click routing interface.
- Complete automated pytest testing suite.
